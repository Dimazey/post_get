Запускаете сервер album_server.py
Для поиска альбомов исполнителя в браузере набираете: localhost:8080/albums/название исполнителя
Для сохранения в базе нового альбома с утилитой HTTP запрос вроде:
http -f POST http://localhost:8080/albums year="1950" artist="New Artist" genre="Rock" album="Super" 

сервер проверит год - число ли это и не пустая строка 
Проверяется не пусты ли параметры artist и album

Если лень печатать HTTP запрос пользуйтесь формой в файле save_album.html
